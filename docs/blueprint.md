# **App Name**: AI Guardian

## Core Features:

- Incoming Call Analysis: Analyze incoming calls to determine the risk level and identify potential scam attempts.
- AI Agent Takeover: Allows the AI agent to take over interactions during suspected scam calls, engaging with scammers and extracting intelligence. The Agent will use a tool to help it decide when it should offer particular information to the other participant.
- Scam Intelligence Panel: Provides real-time detection of entities such as UPI IDs, bank accounts, and suspicious links during the AI-handled calls.
- Call History & Scam Log: Maintains a call log with labels for safe, scam, and AI-handled calls, providing users with a clear overview of their call history.
- Evidence and Report View: Displays a full conversation transcript and extracted scammer data, with options to download and share for reporting purposes.
- User Profile Setup: Enables users to set up a profile with their name, phone number, optional photo, and language selection.
- Settings/Guardian Controls: Allows users to configure AI protection sensitivity and behavior, including polite, confused, and assertive modes, with manual approval options.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for trust and security.
- Background color: Light cyan (#E0FFFF) to maintain a light and calming feel.
- Accent color: Medium slate blue (#7B68EE) for highlights and active elements. The tint is slightly darker than the primary color, providing emphasis.
- Headline font: 'Space Grotesk' sans-serif font for headlines and short body text; it gives a computerized and scientific feel.
- Body font: 'Inter' sans-serif font for body text, paired with Space Grotesk, gives a machined and neutral look.
- Use flat, clear icons to represent call status, risk levels, and AI actions.
- Subtle animations that react to call activity, such as a pulsing glow around the Guardian Sphere.
